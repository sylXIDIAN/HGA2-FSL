# HGA2-FSL

The code in this repository implements the "HGA2-FSL: A Heterogeneous Graph Aware-Aggregation Driven Few-Shot Learning Network for Hyperspectral Image Change Detection".

We run this code with python 3.8, torch==1.11.0, and CUDA==11.3.

Please kindly cite our paper if this code is useful and helpful for your research.

S. Li, X. Xie, X. Zhao and X. Wang, "HGA2-FSL: A Heterogeneous Graph Aware-Aggregation Driven Few-Shot Learning Network for Hyperspectral Image Change Detection," in IEEE Transactions on Geoscience and Remote Sensing, vol. 64, pp. 1-14, 2026, Art no. 5508314, doi: 10.1109/TGRS.2026.3673671.
